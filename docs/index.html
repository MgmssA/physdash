Allgemeine Scheitelpunktform

Die parametrisierte Scheitelpunktform einer Normalparabel ist

y = (x - a)^2 + b.

Dabei sind a und b Parameter, die die Position der Parabel bestimmen. Aus dieser Gleichung kannst Du die Koordinaten des Scheitelpunktes direkt als S(a|b) ablesen. Spiel jetzt mit den Parametern und beobachte, was passiert.

import {tex} from "@observablehq/tex"

function parabola(x, a, b) {
    return (Math.pow((x - a), 2) + b);
}

rangeX = {
    let x = [];
    for (let i = -100; i <= 100; i++) {
        x.push(i / 10);
    }
    return x
}

dynamicVertex = `S(${parameterA})|${parameterB})`

normalData = Array.from(rangeX, (x) => {
        return {x: x, y: parabola(x, 0, 0)};
    })

dynamicData = Array.from(rangeX, (x) => {
        return {x: x, y: parabola(x, parameterA, parameterB)};
    })
Spezifische Scheitelpunktform

Wenn Du die Werte der Parameter einstellst, ergibt sich für die Scheitelpunktform der Normalparabel:

tex.block`y = (x - ${parameterA})^2 + ${parameterB}.`

Hierbei kannst Du den Scheitelpunkt direkt ablesen als

tex.block`S(${parameterA}|${parameterB}).`
Parameter 
viewof parameterA = Inputs.range(
    [-5, 5], 
    {value: 0, step: 0.1, label: "Parameter a:", width: "80%"}
)
viewof parameterB = Inputs.range(
    [-5, 5], 
    {value: 0, step: 0.1, label: "Parameter b:", width: "80%"}
